# Setup nmap (Linux users) 
1. Install nmap INSIDE YOUR SYSTEM GLOBALLY. (Do this in terminal)
> nmap installation is to be done in system level. 
```bash
$ sudo apt update
$ sudo apt install nmap
```
2. Confirm the installation and path: 
```bash
$ which nmap # should show /usr/bin/nmap
$ nmap --version # should show the nmap version
```
